Features

Everything your CLI, UI, and documentation workstreams need in one Bun-powered monorepo.

Material docs

A responsive two-column layout with sticky sidebar navigation, gradient surfaces, and elevated cards that make this feel like a modern Expo Router companion instead of a raw repo view.

Roadmap & plan

The CLI architecture, workspace structure, and future features derive from Plan.md, ensuring the documentation iterates the same narrative as the codebase.

Security & automation

Bun audit, Dependabot, lint, typecheck, and Expo builds run on every push through the security workflow. Claude validates changes before they reach main.

Live changelog

Release highlights fetched dynamically from Markdown, keeping the page in sync with git history.

Security

Bun audit, Dependabot, Claude hooks, and a documented disclosure path.

Every change touches Bun audit, lint, typecheck, and build commands before merging into main. Dependabot monitors bun.lockb and GitHub Actions weekly for vulnerabilities. See Security.md for complete guardrails and security reporting instructions.
